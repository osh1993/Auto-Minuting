    `gradle/libs.versions.toml`에 추가:
    ```toml
    [versions]
    firebaseAiLogic = "16.0.0"
    firebaseBom = "33.12.0"
    googleServices = "4.4.2"

    [libraries]
    firebase-bom = { group = "com.google.firebase", name = "firebase-bom", version.ref = "firebaseBom" }
    firebase-ai-logic = { group = "com.google.firebase", name = "firebase-ai", version.ref = "firebaseAiLogic" }

    [plugins]
    google-services = { id = "com.google.gms.google-services", version.ref = "googleServices" }
    ```

    주의: Firebase AI Logic SDK(`firebase-ai`)는 Gemini API를 Firebase 프로젝트 경유로 호출하는 공식 SDK이다. `google-services.json` 파일이 `app/` 디렉토리에 있어야 한다. 없는 경우에도 컴파일은 되도록 `google-services` 플러그인 적용을 조건부로 처리하거나, 빈 `google-services.json` 대안을 사용한다.

    그러나 Firebase 프로젝트 설정이 사용자 환경에 의존하므로, **대안 접근으로 `com.google.ai.client.generativeai:generativeai` (Google AI Client SDK for Android)를 사용한다.** 이 SDK는 Firebase 없이 API 키만으로 Gemini API를 직접 호출할 수 있다.

    실제 추가할 의존성:
    ```toml
    [versions]
    generativeai = "0.9.0"

    [libraries]
    generativeai = { group = "com.google.ai.client.generativeai", name = "generativeai", version.ref = "generativeai" }
    ```
