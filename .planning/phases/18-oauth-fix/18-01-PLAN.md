# Phase 18 Plan 01: Google OAuth Web Client ID 설정 UI 추가

## Objective

Google OAuth Web Client ID를 설정 화면에서 입력할 수 있게 하여, BuildConfig에 의존하지 않고도 OAuth 로그인이 가능하도록 한다.

## Context

- @app/src/main/java/com/autominuting/data/security/SecureApiKeyRepository.kt
- @app/src/main/java/com/autominuting/data/auth/GoogleAuthRepository.kt
- @app/src/main/java/com/autominuting/presentation/settings/SettingsScreen.kt
- @app/src/main/java/com/autominuting/presentation/settings/SettingsViewModel.kt

## Tasks

### Task 1: SecureApiKeyRepository에 Google OAuth Client ID 저장/조회 메서드 추가
type="auto"

SecureApiKeyRepository에 saveGoogleOAuthClientId(), getGoogleOAuthClientId(), clearGoogleOAuthClientId() 메서드를 추가한다.

**Done criteria:**
- KEY_GOOGLE_OAUTH_CLIENT_ID 상수 추가
- save/get/clear 3개 메서드 추가

### Task 2: GoogleAuthRepository에서 SecureApiKeyRepository 우선 참조
type="auto"

GoogleAuthRepository.signIn()에서 BuildConfig 대신 SecureApiKeyRepository.getGoogleOAuthClientId()를 먼저 확인하고, 없으면 BuildConfig로 폴백한다.

**Done criteria:**
- GoogleAuthRepository 생성자에 SecureApiKeyRepository 주입
- signIn()에서 SecureApiKeyRepository -> BuildConfig 순서로 Client ID 조회
- 둘 다 비어있을 때만 에러 표시

### Task 3: SettingsScreen에 Google OAuth Client ID 입력 필드 추가
type="auto"

설정 화면의 Google 계정(OAuth) 섹션에 Client ID 입력 필드를 추가한다. SettingsViewModel에 관련 메서드를 추가한다.

**Done criteria:**
- SettingsViewModel에 saveOAuthClientId, hasOAuthClientId, clearOAuthClientId 추가
- GoogleAccountSection 위에 OAuth Client ID 입력 UI 표시
- 저장/삭제 버튼 동작

## Verification

- SecureApiKeyRepository에 3개 메서드 존재
- GoogleAuthRepository.signIn()이 SecureApiKeyRepository 우선 참조
- SettingsScreen OAuth 섹션에 Client ID 입력 필드 존재
- 빌드 오류 없음 (구문 검증)

## Success Criteria

사용자가 설정 화면에서 Google OAuth Web Client ID를 입력하면, BuildConfig 없이도 Google 로그인이 가능하다.
