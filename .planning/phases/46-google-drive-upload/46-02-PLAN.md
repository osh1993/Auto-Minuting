`DriveUploadWorker.kt` 신규 생성. TranscriptionTriggerWorker/MinutesGenerationWorker의 @HiltWorker 패턴을 그대로 따른다.

**클래스 선언:**
```kotlin
@HiltWorker
class DriveUploadWorker @AssistedInject constructor(
    @Assisted appContext: Context,
    @Assisted workerParams: WorkerParameters,
    private val googleAuthRepository: GoogleAuthRepository,
    private val driveUploadRepository: DriveUploadRepository,
    private val userPreferencesRepository: UserPreferencesRepository
) : CoroutineWorker(appContext, workerParams)
```

**doWork() 구현 — 아래 순서로 처리:**

1. **재시도 횟수 제한:** `if (runAttemptCount >= MAX_ATTEMPTS) return Result.failure(workDataOf(KEY_ERROR to "최대 재시도 초과"))`

2. **inputData 읽기:**
   - `filePath = inputData.getString(KEY_FILE_PATH) ?: return Result.failure()`
   - `fileType = inputData.getString(KEY_FILE_TYPE) ?: return Result.failure()`

3. **Drive access token 확인:**
   - `val accessToken = googleAuthRepository.getDriveAccessToken()`
   - `if (accessToken.isNullOrBlank()) return Result.failure(workDataOf(KEY_ERROR to "Drive access token 없음. 설정에서 Drive 연동 필요."))`
   - 토큰 없음은 재시도로 해결 안 됨 → 즉시 failure

4. **폴더 ID 조회:**
   - `TYPE_TRANSCRIPT` → `userPreferencesRepository.getDriveTranscriptFolderIdOnce()`
   - `TYPE_MINUTES` → `userPreferencesRepository.getDriveMinutesFolderIdOnce()`
   - `if (folderId.isBlank()) { Log.d(TAG, "Drive 폴더 미설정 — 건너뜀"); return Result.success() }` — 오류 아님

5. **파일 존재 확인:**
   - `val file = File(filePath)`
   - `if (!file.exists()) return Result.failure(workDataOf(KEY_ERROR to "파일 없음: $filePath"))`

6. **업로드 실행:**
   - `mimeType = "text/plain"` (txt/md 모두)
   - `val uploadResult = driveUploadRepository.uploadFile(accessToken, file.name, mimeType, file.readBytes(), folderId)`

7. **결과 처리:**
   - `uploadResult.isSuccess` → `Log.d(...) ; Result.success()`
   - `uploadResult.exceptionOrNull() is UnauthorizedException` → `Result.failure(workDataOf(KEY_ERROR to "Drive 토큰 만료 — 재인증 필요"))` (재시도 없음)
   - 그 외 → `Log.w(...) ; Result.retry()`

**companion object:**
```kotlin
companion object {
    const val KEY_FILE_PATH = "driveFilePath"
    const val KEY_FILE_TYPE = "driveFileType"
    const val KEY_MEETING_ID = "meetingId"
    const val KEY_ERROR = "driveError"
    const val TYPE_TRANSCRIPT = "transcript"
    const val TYPE_MINUTES = "minutes"
    private const val MAX_ATTEMPTS = 3
    private const val TAG = "DriveUploadWorker"
}
```
