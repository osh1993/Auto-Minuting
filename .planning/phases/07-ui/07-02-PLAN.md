- 기존 `meetings` StateFlow를 검색 로직으로 교체:
```kotlin
val meetings: StateFlow<List<Meeting>> = _searchQuery
    .debounce(300) // per specifics: 300ms debounce
    .flatMapLatest { query ->
        if (query.isBlank()) {
            meetingRepository.getMeetings()
        } else {
            meetingRepository.searchMeetings(query.trim())
        }
    }
    .map { list -> list.sortedByDescending { it.minutesPath != null } }
    .stateIn(
        scope = viewModelScope,
        started = SharingStarted.WhileSubscribed(5_000),
        initialValue = emptyList()
    )
```
