2. **TranscriptsViewModel 수정**:
   - `private val promptTemplateRepository: PromptTemplateRepository` 주입 추가
   - `private val minutesRepository: MinutesRepository` 주입 추가
   - `private val meetingDao: MeetingDao` 주입 추가 (또는 MeetingRepository 사용)
   - `val templates: StateFlow<List<PromptTemplate>>` — promptTemplateRepository.getAll() + ensureDefaultTemplates()
   - `private val _isManualGenerating = MutableStateFlow(false)` + `val isManualGenerating: StateFlow<Boolean>`
   - `private val _manualGenerationResult = MutableSharedFlow<ManualGenerationResult>()` + `val manualGenerationResult: SharedFlow<ManualGenerationResult>` (sealed class: Success(meetingId), Error(message))
   - `fun generateManualMinutes(meetingId: Long, templateId: Long?, customPrompt: String?)`:
     ```
     viewModelScope.launch {
       _isManualGenerating.value = true
       try {
         // 1. Meeting 조회, transcriptPath 가져오기
         val meeting = meetingRepository.getMeetingById(meetingId) 또는 meetingDao.getMeetingByIdOnce(meetingId)
         val transcriptText = File(meeting.transcriptPath).readText()

         // 2. 프롬프트 결정
         val prompt = if (templateId != null) {
           promptTemplateRepository.getById(templateId)?.promptText
         } else { customPrompt }

         // 3. 상태 업데이트: GENERATING_MINUTES
         meetingDao.updatePipelineStatus(meetingId, GENERATING_MINUTES, null, now)

         // 4. 회의록 생성 (커스텀 프롬프트 사용)
         val result = minutesRepository.generateMinutes(transcriptText, customPrompt = prompt)

         // 5. 성공: 파일 저장 + DB 업데이트 (COMPLETED)
         val minutesText = result.getOrThrow()
         val minutesDir = File(context.filesDir, "minutes")
         minutesDir.mkdirs()
         val file = File(minutesDir, "${meetingId}.md")
         file.writeText(minutesText)
         meetingDao.updateMinutes(meetingId, file.absolutePath, COMPLETED, now)

         _manualGenerationResult.emit(ManualGenerationResult.Success(meetingId))
       } catch (e: Exception) {
         meetingDao.updatePipelineStatus(meetingId, FAILED, e.message, now)
         _manualGenerationResult.emit(ManualGenerationResult.Error(e.message ?: "알 수 없는 오류"))
       } finally {
         _isManualGenerating.value = false
       }
     }
     ```
   - `sealed class ManualGenerationResult { data class Success(val meetingId: Long) : ..., data class Error(val message: String) : ... }`
