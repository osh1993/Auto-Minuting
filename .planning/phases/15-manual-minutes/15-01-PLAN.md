3. **DAO 생성** — `data/local/dao/PromptTemplateDao.kt`:
   - `@Query("SELECT * FROM prompt_templates ORDER BY isBuiltIn DESC, name ASC") fun getAllTemplates(): Flow<List<PromptTemplateEntity>>` (기본 제공 먼저, 그 다음 이름순)
   - `@Query("SELECT * FROM prompt_templates WHERE id = :id") suspend fun getById(id: Long): PromptTemplateEntity?`
   - `@Insert suspend fun insert(entity: PromptTemplateEntity): Long`
   - `@Update suspend fun update(entity: PromptTemplateEntity)`
   - `@Query("DELETE FROM prompt_templates WHERE id = :id AND isBuiltIn = 0") suspend fun delete(id: Long)` (isBuiltIn 보호)
   - `@Query("SELECT COUNT(*) FROM prompt_templates") suspend fun count(): Int`

   - `abstract fun promptTemplateDao(): PromptTemplateDao` 추가

   - `.addMigrations(AppDatabase.MIGRATION_1_2, AppDatabase.MIGRATION_2_3)` 으로 변경
   - `@Provides fun providePromptTemplateDao(db: AppDatabase): PromptTemplateDao = db.promptTemplateDao()` 추가

   - `suspend fun delete(id: Long)`
