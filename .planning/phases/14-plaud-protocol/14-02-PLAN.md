    사용자가 Plaud Note 녹음기 근처에서 nRF Connect 앱으로 BLE 스캔 및 GATT 탐색을 수행한다:

    9. 아래 형식으로 발견된 서비스/특성 정보를 텍스트로 입력:

    ```
    서비스 1: [UUID] - [Standard/Custom]
      특성 1-1: [UUID] - Properties: [Read/Write/Notify/...]
      특성 1-2: [UUID] - Properties: [...]
    서비스 2: [UUID] - [Standard/Custom]
      특성 2-1: [UUID] - Properties: [...]
    ```

    11. Bluetooth OFF -> ON
    12. Plaud 앱으로 파일 동기화 수행
    13. `adb pull /data/misc/bluetooth/logs/btsnoop_hci.log ./captures/` 로 로그 추출
