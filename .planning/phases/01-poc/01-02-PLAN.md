2. **Galaxy AI 서드파티 접근 조사** (galaxy-ai-investigation.md, per D-05):
   모든 경로를 조사하고 결과를 기록한다:

   a) SpeechRecognizer 온디바이스 경로:
      - Android SpeechRecognizer API에서 `createOnDeviceSpeechRecognizer(context)` (API 31+) 확인
      - Samsung 기기에서 기본 STT 엔진이 무엇인지 확인하는 방법 기록:
        `Settings > General management > Keyboard list and default > Speech services`
      - Samsung 기기에서 SpeechRecognizer가 Galaxy AI 엔진을 사용하는지 확인 필요 (실기기 테스트 항목으로 기록)
      - 제한사항: SpeechRecognizer는 스트리밍 입력만 지원 (파일 입력 미지원). 파일 재생 + 마이크 루프백은 비현실적

   b) Samsung 녹음앱/노트앱 Intent 경로:
      - Samsung Voice Recorder 패키지명: com.sec.android.app.voicenote
      - Intent 액션 조사: ACTION_TRANSCRIBE 또는 유사한 Intent가 존재하는지 검색
      - Samsung Notes 전사 기능 트리거 Intent 조사
      - 방법: `adb shell dumpsys package com.sec.android.app.voicenote | grep -i intent` 또는 매니페스트 분석
      - adb 없이도: Samsung Developer 문서에서 해당 Intent 존재 여부 확인

   c) Samsung SDK/API 경로:
      - Samsung Developer Portal(developer.samsung.com) 검색: "transcript", "speech", "AI"
      - Samsung Galaxy AI SDK 존재 여부 확인
      - 결과 기록: API 존재/미존재, 접근 조건

   d) 각 경로별 결론:
      - ACCESSIBLE / NOT_ACCESSIBLE / REQUIRES_DEVICE_TEST 중 하나로 판정

3. **ML Kit GenAI SpeechRecognizer 평가** (mlkit-evaluation.md):
   - 공식 문서 기반 평가: https://developers.google.com/ml-kit/genai/speech-recognition/android
   - 한국어(ko-KR) 지원 확인: Basic 모드와 Advanced 모드 구분
   - 파일 입력 지원 확인: AudioSource.fromFileDescriptor(fd) API
   - Samsung 기기 호환성 분석:
     - AICore 앱 의존성: Samsung 기기에서 AICore 설치 가능 여부
     - Advanced 모드: Pixel 10 전용 -> Samsung 미지원
     - Basic 모드: 범용 가능성 있으나 실기기 테스트 필요
   - 모델 다운로드 크기 및 초기화 시간 (alpha이므로 정보 제한적)
   - 섹션: ## 개요, ## 한국어 지원, ## 파일 입력, ## Samsung 호환성, ## 리스크, ## 평가(VIABLE/NOT_VIABLE/NEEDS_DEVICE_TEST)

4. **Whisper 온디바이스 평가** (whisper-evaluation.md, per D-07):
   - whisper.cpp GitHub 저장소(https://github.com/ggml-org/whisper.cpp) 분석
   - Android 빌드 방법: whisper.android 예제 앱 (Kotlin)
   - 모델 선택 (Claude's Discretion): small 모델 권장
     - tiny(~75MB): 한국어 정확도 부족 예상
     - base(~150MB): 한국어 정확도 불충분 가능
     - small(~500MB): 정확도와 속도 균형, PoC 권장
   - 한국어 성능: S24 Ultra에서 distil-small.en 2x 실시간 (영어 기준, 한국어는 다를 수 있음)
   - Hallucination 리스크: VAD 전처리, temperature 조정으로 완화
   - Android 통합 복잡도: JNI 빌드, NDK 필요
   - 섹션: ## 개요, ## 모델 선택, ## Android 빌드, ## 한국어 성능 예상, ## Hallucination 대응, ## 평가(VIABLE/NOT_VIABLE)
