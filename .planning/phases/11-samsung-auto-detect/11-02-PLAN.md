    - suspend fun diffSnapshots(before, after): List<MediaStoreEntry>
      - dateAdded 기준으로 after에만 있는 새 파일 목록 반환
    - data class MediaStoreEntry(id, displayName, relativePath, dateAdded, mimeType, size, ownerPackageName)
    - fun formatReport(audioEntries, textEntries, diff): String
      - 사람이 읽기 쉬운 형태로 결과 포맷팅

    - "검증 시작 (Before 스냅샷)" 버튼 추가: 현재 MediaStore 상태를 저장
    - "검증 완료 (After 스냅샷 + Diff)" 버튼 추가: 새 스냅샷을 찍고 Before와 비교하여 새로 추가된 파일 목록 표시
