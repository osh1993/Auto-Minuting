**1. 파일 종류 아이콘 (CARD-01)**
제목 왼쪽에 아이콘을 추가한다. 제목 Row 내부, Text 앞에 Icon을 배치:
- `meeting.audioFilePath.isNotBlank()` 이면 `Icons.Default.AudioFile` (음성 파일)
- `meeting.audioFilePath.isBlank()` 이면 `Icons.Default.TextSnippet` (텍스트 공유)
- 아이콘 크기: 20.dp, 색상: MaterialTheme.colorScheme.onSurfaceVariant
- 아이콘과 제목 사이 간격: Spacer(Modifier.width(8.dp))
- contentDescription: 음성이면 "음성 파일", 텍스트면 "텍스트 공유"
- import 추가: `import androidx.compose.material.icons.filled.AudioFile`, `import androidx.compose.material.icons.filled.TextSnippet`, `import androidx.compose.foundation.layout.width`, `import androidx.compose.foundation.layout.size`, `import androidx.compose.ui.unit.dp` (이미 import된 것은 생략)

**2. 전사 상태 배지 (CARD-02)**
녹음 시각 Row 아래에 새 Row를 추가하여 상태 배지들을 배치한다.
- `meeting.transcriptPath != null` 이면: SuggestionChip, label="전사 완료", containerColor=primaryContainer, labelColor=onPrimaryContainer
- `meeting.transcriptPath == null` 이면: SuggestionChip, label="전사 미완료", containerColor=surfaceVariant, labelColor=onSurfaceVariant
- 단, TRANSCRIBING 상태일 때는 "전사 중" 표시 (tertiaryContainer/onTertiaryContainer)

**3. 회의록 상태 배지 (CARD-03)**
전사 상태 배지와 같은 Row에 나란히 배치한다 (horizontalArrangement = spacedBy(8.dp)):
- `meeting.minutesPath != null` 이면: SuggestionChip, label="회의록 완료", containerColor=primaryContainer, labelColor=onPrimaryContainer
- `meeting.minutesPath == null` 이면: SuggestionChip, label="회의록 미작성", containerColor=surfaceVariant, labelColor=onSurfaceVariant
- 단, GENERATING_MINUTES 상태일 때는 "회의록 생성 중" (tertiaryContainer/onTertiaryContainer)

**배지 Row 구조:**
녹음 시각 Row와 기존 PipelineStatusChip Row 사이, Spacer(height=4.dp) 후에 배치:

**기존 PipelineStatusChip 제거:**
기존 PipelineStatusChip은 AUDIO_RECEIVED/TRANSCRIBING/TRANSCRIBED/GENERATING_MINUTES/COMPLETED/FAILED를 한 칩으로 표시하던 것인데, 새 배지가 전사/회의록 상태를 각각 표시하므로 기존 PipelineStatusChip 호출을 제거한다. 대신 FAILED 상태일 때만 별도로 에러 칩을 표시한다:
- `meeting.pipelineStatus == PipelineStatus.FAILED` 이면 배지 Row에 추가: SuggestionChip, label="오류", containerColor=errorContainer, labelColor=onErrorContainer

```kotlin
@Composable
private fun FileTypeIcon(meeting: Meeting) {
    val isAudio = meeting.audioFilePath.isNotBlank()
    Icon(
        imageVector = if (isAudio) Icons.Default.AudioFile else Icons.Default.TextSnippet,
        contentDescription = if (isAudio) "음성 파일" else "텍스트 공유",
        modifier = Modifier.size(20.dp),
        tint = MaterialTheme.colorScheme.onSurfaceVariant
    )
}

@Composable
private fun TranscriptionStatusBadge(meeting: Meeting) {
    // TRANSCRIBING -> "전사 중" (tertiary), transcriptPath != null -> "전사 완료" (primary), else -> "전사 미완료" (surface)
}

@Composable
private fun MinutesStatusBadge(meeting: Meeting) {
    // GENERATING_MINUTES -> "회의록 생성 중" (tertiary), minutesPath != null -> "회의록 완료" (primary), else -> "회의록 미작성" (surface)
}
```
