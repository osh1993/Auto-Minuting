    **AudioCollectionService.kt** (패키지: `com.autominuting.service`):
    - `@AndroidEntryPoint class AudioCollectionService : Service()`
    - Hilt 주입: `@Inject lateinit var audioRepository: AudioRepository`
    - Hilt 주입: `@Inject lateinit var workManager: WorkManager`
    - 필드: `private val serviceScope = CoroutineScope(SupervisorJob() + Dispatchers.IO)`
    - 필드: `private var collectionJob: Job? = null`

    - `override fun onCreate()`: NotificationChannel 생성
      - channelId = `"audio_collection_channel"`
      - channelName = `"오디오 수집"`
      - importance = `NotificationManager.IMPORTANCE_LOW`

    - `override fun onStartCommand(intent: Intent?, flags: Int, startId: Int): Int`:
      - intent action 분기: `ACTION_START` / `ACTION_STOP`
      - ACTION_START:
        1. `createNotification("오디오 수집 대기 중...")` 으로 알림 생성
        2. `ServiceCompat.startForeground(this, NOTIFICATION_ID, notification, ServiceInfo.FOREGROUND_SERVICE_TYPE_CONNECTED_DEVICE)`
        3. `collectionJob = serviceScope.launch { collectAudio() }`
      - ACTION_STOP: `stopSelf()`
      - `return START_STICKY`

    - `private suspend fun collectAudio()`:
      1. `audioRepository.startAudioCollection().collect { filePath -> }`
      2. collect 안에서: `updateNotification("오디오 저장 완료: ${File(filePath).name}")`
      3. collect 안에서: `triggerTranscriptionPipeline(filePath)` 호출 (per AUD-03)

    - `private fun triggerTranscriptionPipeline(filePath: String)`:
      - `val inputData = workDataOf("audioFilePath" to filePath)`
      - `OneTimeWorkRequestBuilder<TranscriptionTriggerWorker>()` 빌드
      - constraints: `setRequiresBatteryNotLow(true)`
      - backoff: `BackoffPolicy.EXPONENTIAL, WorkRequest.MIN_BACKOFF_MILLIS, TimeUnit.MILLISECONDS`
      - `workManager.enqueueUniqueWork("transcription_${System.currentTimeMillis()}", ExistingWorkPolicy.KEEP, workRequest)`

    - `private fun createNotification(text: String): Notification`:
      - `NotificationCompat.Builder(this, CHANNEL_ID)` 사용
      - `.setSmallIcon(R.drawable.ic_launcher_foreground)`
      - `.setContentTitle("Auto Minuting")`
      - `.setContentText(text)`
      - `.setOngoing(true)`

    - `private fun updateNotification(text: String)`: NotificationManager로 알림 업데이트

    - `override fun onDestroy()`:
      - `collectionJob?.cancel()`
      - `serviceScope.cancel()`
      - `super.onDestroy()`

    - `override fun onBind(intent: Intent?): IBinder? = null`

    - companion object:
      - `const val NOTIFICATION_ID = 1001`
      - `const val CHANNEL_ID = "audio_collection_channel"`
      - `const val ACTION_START = "com.autominuting.action.START_COLLECTION"`
      - `const val ACTION_STOP = "com.autominuting.action.STOP_COLLECTION"`
