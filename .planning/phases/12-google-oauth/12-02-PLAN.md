Google 인증 인프라와 OAuth 기반 Gemini 엔진을 구현한다.

1. **AuthState.kt 생성** (`data/auth/AuthState.kt`):
   ```kotlin
   package com.autominuting.data.auth

   // 인증 모드
   enum class AuthMode { API_KEY, OAUTH }

   // Google 인증 상태
   sealed interface AuthState {
       data object NotSignedIn : AuthState
       data class SignedIn(
           val displayName: String,
           val email: String,
           val photoUrl: String? = null
       ) : AuthState
       data object Loading : AuthState
       data class Error(val message: String) : AuthState
   }
   ```

2. **GoogleAuthRepository.kt 생성** (`data/auth/GoogleAuthRepository.kt`):
   - `@Singleton`, `@Inject constructor(@ApplicationContext context: Context)`
   - `WEB_CLIENT_ID`는 BuildConfig에서 읽음 (build.gradle.kts에 추가 필요)
   - **signIn(activityContext: Context)**: Credential Manager API로 Google Sign-In 실행
     - `GetGoogleIdOption.Builder().setFilterByAuthorizedAccounts(false).setServerClientId(WEB_CLIENT_ID).build()`
     - `CredentialManager.create(context).getCredential(activityContext, request)`
     - 성공 시 GoogleIdTokenCredential에서 displayName, email 추출하여 DataStore에 저장
   - **authorize(activity: Activity)**: AuthorizationClient로 access token 획득
     - 스코프: 먼저 `Scope("https://www.googleapis.com/auth/generative-language.retriever")` 시도
     - 실패 시 `Scope("https://www.googleapis.com/auth/cloud-platform")` 폴백
     - `Identity.getAuthorizationClient(activity).authorize(request)`
     - 성공 시 access token을 메모리 캐시에 보관
     - `hasResolution()` 시 PendingIntent 반환 (UI 레이어에서 launcher로 처리)
   - **getAccessToken()**: 캐시된 access token 반환 (없으면 null)
   - **signOut()**: 로그인 상태 초기화, DataStore에서 사용자 정보 삭제
   - **isSignedIn()**: DataStore 기반 로그인 상태 확인 (Boolean)
   - **authState**: StateFlow<AuthState> — 현재 인증 상태 관찰용
   - 주의: Credential Manager는 Activity Context 필요 — Repository는 sign-in 결과만 처리, 실제 getCredential 호출은 ViewModel이 Activity context를 전달하는 패턴

3. **BearerTokenInterceptor.kt 생성** (`data/minutes/BearerTokenInterceptor.kt`):
   - OkHttp Interceptor 구현
   - `tokenProvider: suspend () -> String?` 람다를 생성자로 받음
   - intercept() 에서 `runBlocking { tokenProvider() }` 호출하여 토큰 획득
   - `Authorization: Bearer $token` 헤더 추가
   - 토큰이 null이면 헤더 없이 통과 (에러는 서버에서 반환)

4. **GeminiOAuthEngine.kt 생성** (`data/minutes/GeminiOAuthEngine.kt`):
   - `@Singleton`, `@Inject constructor(geminiRestApiService: GeminiRestApiService, googleAuthRepository: GoogleAuthRepository)`
   - `MinutesEngine` 인터페이스 구현
   - `generate()`: 리서치의 GeminiOAuthEngine 골격 코드 참조 — GeminiEngine과 동일한 프롬프트 사용 (GeminiEngine.STRUCTURED_PROMPT 등을 MinutesEngine의 companion object나 별도 상수 객체로 공유)
   - `isAvailable()`: `googleAuthRepository.isSignedIn()`
   - `engineName()`: `"Gemini 2.5 Flash (OAuth)"`
   - 프롬프트 상수는 GeminiEngine에서 `internal`로 공개하거나, `MinutesPrompts` object로 별도 추출하여 두 엔진이 공유

5. **AuthModule.kt 생성** (`di/AuthModule.kt`):
   - `@Module @InstallIn(SingletonComponent::class) object AuthModule`
   - `@Provides @Singleton @Named("oauth")` fun provideOAuthOkHttpClient: OkHttp 클라이언트에 BearerTokenInterceptor + logging interceptor 추가
   - `@Provides @Singleton` fun provideGeminiRestApiService: Retrofit.Builder().baseUrl("https://generativelanguage.googleapis.com/").client(oauthClient).addConverterFactory(GsonConverterFactory.create()).build().create(GeminiRestApiService::class.java)

6. **RepositoryModule.kt 수정**:
   - 기존 `@Binds bindMinutesEngine` 제거
   - `@Module object` 로 `provideActiveMinutesEngine` 추가:
     - `@Provides @Singleton fun provideActiveMinutesEngine(geminiEngine: GeminiEngine, geminiOAuthEngine: GeminiOAuthEngine, userPreferencesRepository: UserPreferencesRepository): MinutesEngine`
     - 기본값: GeminiEngine (API 키 모드)
     - 런타임 전환은 별도 방법: `Provider<GeminiEngine>`, `Provider<GeminiOAuthEngine>` + DataStore의 authMode 설정으로 선택
     - 실제로는 `MinutesEngineSelector` 클래스를 만들어 MinutesEngine을 구현하고 내부에서 authMode에 따라 위임 패턴 사용

7. **UserPreferencesRepository.kt 수정**:
   - `authMode: Flow<AuthMode>` 추가 (DataStore에 "auth_mode" 키)
   - `setAuthMode(mode: AuthMode)` 추가
   - 기본값: `AuthMode.API_KEY`

   - defaultConfig에 `buildConfigField("String", "GOOGLE_OAUTH_WEB_CLIENT_ID", "\"${localProps.getProperty("GOOGLE_OAUTH_WEB_CLIENT_ID", "")}\"")` 추가
