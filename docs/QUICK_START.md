# Auto Minuting 빠른 시작 가이드

**5분 안에 첫 회의록 만들기**

---

## Step 1 — 앱 설치

### APK 설치 (사이드로딩)

1. APK 파일을 기기로 전달 (Drive, USB, 링크 중 선택)
2. `설정` → `생체 인식 및 보안` → `출처를 알 수 없는 앱 설치` → APK 전달에 사용한 앱 → **허용**
3. 파일 관리자에서 APK 탭 → **설치**

---

## Step 2 — Gemini API 키 입력

1. [aistudio.google.com](https://aistudio.google.com) 접속 → "Get API key" → 키 복사
2. 앱 실행 → `설정` 탭 → **Gemini API 키** → 키 붙여넣기 → 저장

---

## Step 3 — 파일 가져오기

### 음성 파일
- **URL 입력**: `대시보드` 탭 → URL 입력 필드에 음성 파일 주소 붙여넣기 → 다운로드
- **앱에서 공유**: 삼성 녹음앱 또는 Plaud 앱 → 파일 공유 → **Auto Minuting** 선택
- **Plaud 링크**: 카카오톡 등에서 web.plaud.ai 링크 공유 → **Auto Minuting** 선택

### 전사 텍스트 파일 (STT 없이 바로 회의록 생성)
- **텍스트 파일 공유**: 파일 관리자에서 `.txt` 파일 선택 → 공유 → **Auto Minuting** 선택
- **텍스트 직접 공유**: 메모앱·브라우저 등에서 텍스트 선택 → 공유 → **Auto Minuting** 선택

---

## Step 4 — 전사 확인

- 하단 탭 `전사` → 진행 중인 파일 확인 (배지로 상태 표시)
- 완료되면 알림이 옵니다

---

## Step 5 — 회의록 생성

### 자동 모드
`설정` → "자동 회의록 생성" ON → 전사 완료 즉시 자동 생성

### 수동 모드
전사 카드의 **⋮ 메뉴** → **회의록 작성** → 프롬프트 선택 → 생성

---

## Step 6 — 결과 확인

하단 탭 `회의록` → 카드 탭 → Markdown 회의록 전문 확인

회의록 공유: 카드 **⋮ 메뉴** → **공유**

---

## 핵심 팁

| 상황 | 해결 |
|------|------|
| 전사가 너무 느림 | 설정에서 Gemini STT로 변경 |
| 인터넷 없이 전사 | Whisper 모델 다운로드 후 온디바이스 전사 |
| 같은 녹음으로 다른 형식 회의록 | ⋮ 메뉴 → "추가 생성" (기존 보존) |
| API 쿼터 초과 | 대시보드 쿼터 위젯 확인, 내일 자동 초기화 |
